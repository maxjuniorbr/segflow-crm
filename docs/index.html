Qualidade & Segurança

Padrões profissionais de segurança e qualidade de código em toda a aplicação.

Autenticação JWT

Access + Refresh tokens com rotação automática. httpOnly cookies. Rate limiting em rotas de auth.

Validação Zod

Schemas de validação em todas as entradas. Middleware centralizado. Proteção contra mass assignment.

SonarCloud

Análise estática contínua. Zero bugs, zero vulnerabilidades. Quality Gate integrado ao pipeline.

SQL Injection-free

Queries parametrizadas. IDOR protection via broker_id. Sanitização de logs contra injection.

Design System

Tokens CSS, CVA para variantes, cn() para composição. Componentes acessíveis com vitest-axe.

Testes Completos

Unit, integration, functional e security tests. Controllers, use cases e componentes React cobertos.
